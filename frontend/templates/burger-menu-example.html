Burger Menu Example

This is an example of the responsive burger menu implementation.

On desktop, you'll see the horizontal navigation menu.

On mobile/tablet (768px and below), the menu will collapse into a burger button.

Click the burger button to see the full-screen overlay menu with smooth animations.

Features:

  • ✅ Responsive design (desktop horizontal, mobile burger)
  • ✅ Smooth burger-to-cross animation
  • ✅ Full-screen overlay with white background
  • ✅ Centered navigation links
  • ✅ Staggered link animations
  • ✅ Smooth open/close transitions
  • ✅ Keyboard accessibility (ESC to close)
  • ✅ Click outside to close
  • ✅ Prevents body scroll when open
  • ✅ Auto-closes on window resize
